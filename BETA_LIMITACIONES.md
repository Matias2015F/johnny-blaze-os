# MotoGestión — Beta Cerrada

Esta versión es una demo controlada para pruebas internas con talleres seleccionados.

## Qué es

- PWA instalable en Android, desktop y (parcialmente) iPhone
- Sistema de gestión operativa: clientes, motos, órdenes, presupuestos, historial, PDF
- Almacenamiento real en Firebase (datos persisten entre sesiones y dispositivos)
- Funciona offline con sincronización automática al reconectar

## Qué NO incluye esta versión

- **Mercado Pago desactivado** — pagos de suscripción por fuera de la plataforma
- **WhatsApp automático desactivado** — los mensajes se generan pero no se envían automáticamente
- **Panel de administración no incluido**
- **Dominio de producción no modificado** — esta beta corre en URL separada

## Limitaciones conocidas

- Esta es una versión beta, no el producto final
- No usar para datos críticos sin respaldo externo adicional
- El proveedor puede interrumpir el acceso durante el período de prueba
- Las funcionalidades pueden cambiar sin previo aviso

## Uso permitido

- Testing interno con talleres controlados
- Validación del flujo operativo real
- Feedback sobre usabilidad

## Uso NO permitido

- Venta abierta a cualquier taller sin supervisión del administrador
- Datos de producción sin respaldo
- Promesas de disponibilidad continua
