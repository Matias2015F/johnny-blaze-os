## Áreas de análisis

### 1. Posicionamiento

- ¿Qué prometen?
- ¿Para quién hablan?
- ¿Qué problema dicen resolver?
- ¿Cuál es su propuesta única?

### 2. Landing Page

- Hero
- CTA principal
- Navegación
- Jerarquía visual y orden de lectura
- Escaneabilidad
- Responsive

### 3. Marketing

- Oferta y pricing
- Prueba gratuita
- Objeciones abordadas
- Garantías
- Testimonios y casos de éxito
- Señales de autoridad

### 4. SEO

- Title y Meta Description
- H1, H2
- URLs
- Rich snippets / Schema
- Indexabilidad
- Velocidad observable

### 5. UX

- Tiempo hasta comprender el valor
- Claridad de los botones
- Fricción de conversión
- Flujo hacia registro / contacto

### 6. Branding

- Identidad visual (colores, tipografía)
- Consistencia
- Percepción de profesionalismo y confianza

### 7. Diferenciación

Para cada competidor:
- ¿Qué hace mejor que MotoGestión?
- ¿Qué hace MotoGestión mejor que ellos?

¿Qué oportunidad de mercado ninguno está aprovechando?

### 8. Sistema de reputación (crítico)

Buscar explícitamente si existe:
- garantía digital
- historial certificado
- QR de verificación
- comprobante inteligente / trazable
- reputación pública del taller
- ranking o calificaciones visibles
- fidelización o efecto de red

Indicar: EXISTE / NO EXISTE / PARCIAL para cada ítem y para cada competidor.
