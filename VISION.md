# VISION.md — Fleischmann Ingeniería de Soluciones

## Qué somos

Fleischmann Ingeniería de Soluciones diseña Sistemas Operativos de Negocio (SON):
sistemas digitales que eliminan pérdidas, ordenan procesos y generan evidencia
para negocios reales. El software es la consecuencia del análisis, nunca el
punto de partida.

## Arquitectura (decidida, no se discute)

```
Fleischmann Ingeniería de Soluciones   ← Empresa (Nivel 1, no cambia por años)
            │
       Método SON                      ← Metodología interna (Nivel 2, cambia solo con evidencia)
            │
       MotoGestión                     ← Producto (Nivel 3, 100% del foco operativo)
            │
   Talleres de motocicletas           ← Cliente
```

- **Método SON** existe como forma de trabajar. No se vende, no se registra,
  no se publica. Se aplica.
- **MotoGestión** es el SON para talleres de reparación y mantenimiento de
  motocicletas. Es el Caso 001 que valida el método. No es "un software":
  es el sistema operativo del taller.
- Futuros productos (veterinarias, comercios, etc.) son consecuencias de que
  MotoGestión tenga clientes pagos. No son requisitos, no se planifican, no
  se nombran hasta entonces.

## Estrategia en una frase

Pensamos como una empresa capaz de crear múltiples Sistemas Operativos de
Negocio, pero ejecutamos como una empresa con una sola misión: lograr que
MotoGestión sea indispensable para sus primeros clientes.

## Principios

1. El negocio define la tecnología, nunca al revés.
2. La evidencia vale más que las opiniones.
3. Todo aprendizaje se transforma en un activo reutilizable.
4. La simplicidad es una ventaja competitiva.
